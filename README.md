# Student Utility Web App

A comprehensive web application for students featuring calculators, CGPA tracking, daily planning, and more!

## 🌟 Features

- 🔐 User Authentication (Login/Signup)
- 🧮 Scientific Calculator with History
- 🎓 CGPA Calculator with Performance Tracking
- 📅 Daily Planner with Task Management
- 👤 Personalized Dashboard with Motivational Quotes
- 📱 Responsive Design

## 🚀 How to Share This Website

### Option 1: Share Files (Simplest)
1. **Zip the project folder** (`d:\HTML_projects\`)
2. **Share the ZIP file** with others
3. **Recipients should**:
   - Extract the ZIP
   - Open `index.html` in any web browser
   - No internet connection required!

### Option 2: Host on GitHub Pages (Free & Easy)
1. **Create a GitHub repository**
2. **Upload all files** from `d:\HTML_projects\` to the repository
3. **Enable GitHub Pages**:
   - Go to repository Settings → Pages
   - Select "main" branch as source
   - Your site will be live at: `https://yourusername.github.io/repository-name/`
4. **Share the GitHub Pages URL**

### Option 3: Deploy to Netlify (Professional)
1. **Sign up for Netlify** (free tier available)
2. **Drag & drop** the entire `d:\HTML_projects\` folder to Netlify's deploy area
3. **Get a custom URL** like `https://your-app-name.netlify.app`
4. **Share the Netlify URL**

### Option 4: Use Vercel (Modern & Fast)
1. **Sign up for Vercel** (free)
2. **Upload the project folder**
3. **Get instant deployment** with custom domain option
4. **Share the Vercel URL**

### Option 5: Traditional Web Hosting
1. **Purchase web hosting** (Hostinger, Bluehost, etc.)
2. **Upload files via FTP**
3. **Share your domain URL**
